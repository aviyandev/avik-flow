# avik/flow

HTTP lifecycle engine for the Avik framework.

## Responsibilities
- HTTP request abstraction
- HTTP response handling
- Middleware pipeline
- Kernel execution

## Design Rules
- No routing
- No controllers
- No framework glue
- Minimal & stable API

## Stability
Frozen after v1.0.0

## Requirements
- PHP 8.1+
- avik/seed ^1.0
- avik/essence ^1.0
